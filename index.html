实验报告 — 答辩总结
一、项目概述

基于机器学习的密码攻击检测与加密算法自适应优化系统
面向隐私保护场景的攻击检测、联邦训练与自适应加密演示系统。

二、技术路线

数据处理 → 联邦训练 → 攻击检测 → 自适应优化 → 安全防护 → 实验报告

三、创新点
  1. 攻击检测结果与加密策略优化联动展示。
  2. 联邦学习多机构节点训练演示。
  3. Paillier / AES 加密对比。
  4. Q-learning 风险驱动参数调整。
  5. TraceId / 慢接口 / 限流 / 安全事件看板闭环。
四、系统边界说明

当前系统为大创项目实验原型，部分能力为模拟、演示或预留设计。联邦训练为单机模拟流程；Paillier / AES 对比来自当前后端接口；Q-learning 用于展示参数调整流程；安全防护模块不代表完整企业级安全运营平台。
