实验报告 - 项目技术路线与答辩总结
项目概述

本系统是面向密码攻击检测场景的大创原型系统，重点展示公开数据集接入、联邦节点划分、攻击检测、参数优化和安全可观测链路。

技术路线

公开数据集 → 特征提取 → 四节点划分 → 联邦训练 → 攻击检测 → 参数优化 → 安全事件追踪。

当前边界

当前为 Flask + SQLite 的原型系统；联邦训练在本地运行四节点流程；安全防护模块展示已接入的请求追踪、慢接口、限流和事件查询能力。

模块能力表
模块当前能力展示方式
数据处理公开数据集检测、处理与四节点划分查看数据源状态、处理数据集并刷新节点状态
联邦训练本地训练基线与四节点聚合流程查看节点样本数、执行一轮联邦训练
攻击检测从已处理数据或上传文件调用检测接口展示风险分数、风险等级和攻击类型
自适应优化根据接口返回展示加密参数调整流程查看当前参数、优化历史和加密对比指标
安全防护TraceId、慢接口、限流和安全事件查询查看安全事件统计和最近事件列表

边界说明：本系统为实验原型，部分能力为模拟或演示，展示结论以当前接口返回和页面流程为准。
