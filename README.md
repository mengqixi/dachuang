
# 密码攻击检测与加密算法自适应优化系统

基于FATE联邦学习框架的密码攻击检测和加密算法自适应优化系统。

## 功能特点

### 1. 数据看板
- 实时显示总攻击数量、检测准确率
- 近6个月攻击检测趋势图表
- 攻击类型分布饼图（暴力破解、侧信道攻击、密文分析、密钥恢复）
- 实时优化状态显示

### 2. 数据准备
- 生成包含敏感字段的模拟数据（手机号、薪资、信用评分）
- 使用Paillier同态加密算法对敏感数据进行加密
- 展示明文数据和加密数据对比

### 3. 模型训练
- 模拟FATE联邦学习训练流程
- 明文训练对比
- 训练日志实时显示
- 训练曲线图表（Accuracy vs Loss）
- 结果对比展示

### 4. 加密算法对比
- 传统加密（AES-256）vs 同态加密（Paillier）
- 数据大小可调节（1-100MB）
- 加密/解密时间对比
- 吞吐量对比
- 安全级别对比
- 联邦学习支持情况对比

### 5. 数据上传与攻击检测
- 支持CSV、JSON文件上传
- 实时攻击检测
- 攻击结果统计（记录数量、异常检测数、准确率）
- 检测结果表格展示

## 使用说明

### 启动服务器

**方法1：使用start_server.py（推荐）**
```bash
python start_server.py
```

**方法2：使用Python内置HTTP服务器**
```bash
python -m http.server 5000
```

**方法3：使用Flask**
```bash
python app.py
```

### 访问系统

打开浏览器，访问：
```
http://127.0.0.1:5000/index.html
```

### 演示数据

系统已包含示例数据：`sample_data.csv`

## 功能详解

### 数据加密与联邦学习

系统使用Paillier同态加密算法确保数据隐私，同时实现多方安全计算：
```
明文 → 加密 → 密态训练 → 密态预测 → 解密
```

### 攻击检测

支持四种主要攻击类型检测：
- 暴力破解攻击
- 侧信道攻击
- 密文分析攻击
- 密钥恢复攻击

### 自适应优化

根据异常检测结果动态调整加密参数：
- 密钥长度（1024/2048/4096位）
- 加密轮数
- 实时优化策略
